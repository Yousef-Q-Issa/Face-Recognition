# Face Recognition System

A Face Recognition System developed using Python, OpenCV, and the face_recognition library. The project detects faces, generates facial encodings, and identifies individuals by comparing unknown faces with a database of known faces.

## Features

* Face detection from images and webcam streams
* Face encoding and feature extraction
* Recognition of known individuals
* Real-time webcam face recognition
* Image-based face recognition
* Configurable recognition settings
* Reusable face encodings for faster performance

## Technologies

* Python
* OpenCV
* face_recognition
* NumPy
* Pickle

## Project Structure

* `encode_faces.py` – Generate and save face encodings
* `recognize_image.py` – Recognize faces from images
* `recognize_webcam.py` – Real-time webcam recognition
* `face_utils.py` – Face processing utilities
* `config.py` – Project configuration settings
